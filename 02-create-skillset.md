# Creating a Skillset

## Type

```http
PUT
```

## URL

```http
https://your-azure-search-service-name.search.windows.net/skillsets/newskillset?api-version=2017-11-11-Preview
```

## Headers

```http
Content-Type : application/json
api-key : <your-azure-search-key>
```
