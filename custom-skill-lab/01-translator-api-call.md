# Calling the Bing Translator API

## Type

```http
POST
```

## URL

```http
https://api.cognitive.microsofttranslator.com/translate?api-version=3.0&from=es&to=en
```

## Headers

```http
Content-Type : application/json
Ocp-Apim-Subscription-Key : your-azure-search-key
```

## Body
```json
[
	{"Text":"Hola Mundo"}
]
```
